TheArticle

Description

  • Contract type: Long term freelance build and maintenance

  • Client: Quad Journal Ltd

  • View at: www.thearticle.com

  • Build completed on: 18 March, 2019 with ongoing upgrades and server maintenance

A non-partisan publishing platform with a strong social media element helping readers get a personalised debate on current affairs without a paywall.

The customer facing web app was built with Ruby on Rails whilst the backend uses a version of Wordpress customised for the editorial team.
