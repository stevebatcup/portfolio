Maawol Online Music Schools

Description

  • Contract type: Personal project

  • View at: www.maawol.com/

  • View source code on Github

  • Build completed on: 21 June, 2020

Earlier in 2020, at the start of the lockdown I started this project to help some struggling friends of mine who are music teachers.

Maawol hosts a library of music lessons using video, audio and text, categorises and groups lessons into courses and helps teachers connect online with their students with the aim of turning them into paying subscribers.

This homepage and blog serves as a brochure for the individual school sites. See this item as an example of a Maawol school.

Unfortunately, lacking marketing resources, this project never really got off of the ground. But it served as a springboard for my up-skilling and my shift into mobile app development.
